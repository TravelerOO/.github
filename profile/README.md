### 사계절을 테마로 여행지에 대한 후기를 남기고 이를 사용자들과 공유합니다.

### 유저 기능
#### 로그인과 회원가입을 통해서 사용자가 여행일지를 작성하고 이를 삭제/수정 할 수 있습니다 . 
#### 사계절에 따라 전국 여행일지을 분류하여서 보여줍니다. 
#### 여행일지를 검색/핕터링 할 수 있습니다. 


# ⚙️ 프로젝트 구조
## 기술스택

## 세부 기술들 

### 인프라 
#### AWS RDS/My sql을 통해서 데이터베이스를 관리합니다. 
#### AWS S3를 통해서 이미지 파일들을 저장/관리합니다. 
#### AWS EC2를 통해서 백엔드 서버를 운영합니다 .

### VCS 
#### Git/Github를 통해서 소스코드를 통합관리합니다. 

### CD
#### Github Actions을 통해서 main 브랜치 업데이트시에, 소스코드를 EC2에 자동배포합니다. 

### 보얀취약점 보완
#### 스프링 시큐리티를 활용하여 유저의 비밀번호를 암호화하여 저장합니다. 
#### jwt를 활용하여 인증된 회원이 인가된 서비스에만 접근하도록 합니다. 
#### 소셜 로그인/회원가입 기능을 통하여 선택적으로 인증/인가를 진행합니다. 
#### 스프링 시큐리티를 적용하여 허용되지 않은 url 접근을 막습니다 .

### ORM/DB

#### SPring DATA JPA를 활용하여서 CRUD를 구현합니다.


### REST API

#### RESTful한 API 설계원칙을 준수하여서 API를 설계합니다.
